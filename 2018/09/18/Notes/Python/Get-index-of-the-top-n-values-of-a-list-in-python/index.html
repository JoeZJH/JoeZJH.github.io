(Python)从数组中返回最大的K个值在数组中的索引

自定义的一个有用的Python方法

从数组中返回最大的K个值在数组中的索引

引用自Stack Overflow

def top_k_indexes(target_list, k, max=True):
    """
    I have a list say a = [5,3,1,4,10]. I need to get a index of top two values of the list viz 5 and 10. Is there a one-liner that python offers for such a case?
    solution: top_k_indexes(target_list=a, k=2, max=True)
    """
    indexes = sorted(range(len(target_list)), key=lambda i: target_list[i], reverse=max)[:k]
    return indexes
